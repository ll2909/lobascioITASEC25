## Code Requirements

The code relies on the following **python3.6+** libs

Packages needed are:
* [PyTorch]
* [Numpy]
* [Pandas]
* [Scikit-learn]
* [LIEF]
* [ember]
* [SHAP]
* [Adversary-Robustness-Toolbox]
